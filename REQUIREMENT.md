# 资产浏览器（Assets）
## 布局
单页应用（无路由切换）。整体为三段式工作台布局：
左侧为窄导航栏（Nav Rail），中间为资源面板与主工作区的组合（可折叠左侧资源面板），右侧为预览窗格（Preview Pane，可按状态显示/隐藏），底部为活动日志抽屉（Activity Dock，可折叠）。

页面主要组成：
1）左侧窄导航栏：提供面板开合与当前页面入口（目前只有 Assets）。
2）左侧资源面板（可折叠）：包含打开文件夹、索引/清单来源信息、统计、集合（Collections）筛选入口等。
3）主工作区：顶部搜索与筛选器；中部资源列表；右侧预览窗格。
4）底部活动日志：展示系统状态、错误与操作反馈。

## 区域1的内容和功能（左侧：导航栏 + 资源面板）
区域目标：选择本地资源根目录，加载/生成索引与元数据，提供全局工具与集合筛选入口。

1. 窄导航栏（Nav Rail）
- Panel 开合：折叠/展开左侧资源面板；折叠后主工作区占用更大宽度。
- Assets 入口：当前应用唯一页面入口（视觉上为 active）。

2. 资源面板（Source Panel）
- 品牌与工具条
  - 收起面板按钮：快速折叠资源面板。
  - 预览模式切换：Click Preview / Hover Preview 两种交互模式。
    - Click Preview：点击图片/视频/模型行打开右侧预览；点击音频行在行内播放。
    - Hover Preview：悬停图片/视频/模型行打开右侧预览；悬停音频行自动播放，移出暂停。
  - 预览锁定（Lock）：锁定后预览窗格保持可见，不随悬停/选择变化而自动隐藏。
  - Reload：重新读取当前文件夹内容（包含索引与元数据的刷新流程）。

- Folder / Manifest 区
  - 显示当前打开的根目录名。
  - 显示索引清单来源与状态（例如使用 asset-browser-index.json 或从 csv/xlsx 生成）。
  - OPEN FOLDER：调用浏览器 File System Access API 选择本地文件夹作为资源根目录。
  - 不支持时提示：当浏览器不支持文件夹读写权限时提示使用 Chrome/Edge。

- 统计信息（Production 统计）
  - Assets：索引中资产总数。
  - Visible：当前过滤条件下可见资产数。
  - Selected：当前选中条目数（支持全选/多选）。
  - Missing：索引路径无法在文件夹中解析到真实文件的数量。
  - Busy 状态：加载/生成/写入时的忙碌提示。

- Collections（集合）列表
  - 展示当前已存在的集合（来自元数据）。
  - 点击集合：作为过滤条件开关（用于在主列表中过滤显示）。
  - 每个集合显示条目数量。

## 区域2的内容和功能（主工作区：筛选 + 列表 + 预览 + 活动）
区域目标：浏览资产、过滤定位、编辑标签与集合、预览媒体、执行常用操作并获得反馈。

1. 顶部搜索与筛选器（Topbar）
- Search：按名称/路径/标签等进行文本匹配搜索（用于快速定位）。
- 布局：Search + Type / Collection / Rating + Reset（图标按钮）同一行；该行下方保留一条分隔线；Tag 过滤器位于分隔线下方并左对齐；Tag 下方不额外显示分隔线。
- Type 筛选：按资产类型过滤（如 image/audio/video/model/document/unknown 等）。
- Tag 筛选：按标签过滤（支持多选；以弹出面板形式选择）。
- Collection 筛选：按集合过滤（支持多选；与左侧集合列表联动）。
- Rating 筛选：按评分过滤（支持多选）。

2. 资产列表（Asset List）
- 列表表头与滚动区：用于承载大量资产行并保持可滚动浏览。
- 行展示字段（核心信息列）
  - Name：资产名称。
  - Type：资产类型（用于预览与筛选）。
  - Rating：评分信息（用于筛选与快速判断）。
  - Path：相对路径/引用路径（指向文件夹内文件）。
  - Status：状态信息（例如 missing 等异常提示）。
  - Tag：标签展示与编辑入口。
  - Actions：常用操作按钮集合。

- 行交互（选择与预览）
  - 单行点击选择；支持全选与多选。
  - 键盘交互：Enter/Space 可激活当前行的主要行为（用于更高效浏览）。
  - 预览触发：遵循当前预览模式（Click/Hover）决定何时在右侧显示预览。

- 标签（Tags）管理
  - 展示部分标签，超过数量时显示溢出计数。
  - 可移除用户标签。
  - 可输入新增标签并写入元数据（asset-browser-metadata.json）。

- 行内音频播放（仅音频类型）
  - 行内播放器：播放/暂停、进度条。
  - 进度条可交互：点击跳转到指定位置播放；在 Hover 模式下支持随鼠标位置连续 seek 并播放。

- Actions（对单个资产的操作）
  - Rename：重命名资产（记录重命名信息并写入元数据）。
  - Download：下载当前资产文件。
  - Favorite / Add to Collection：收藏或加入集合（以元数据形式保存）。
  - Delete：删除资产（对本地文件进行删除并同步状态与日志）。

3. 右侧预览窗格（Preview Pane）
- 显示/隐藏逻辑：由预览模式（Click/Hover）、当前选择、以及 Lock 状态共同决定。
- 空态
  - 未选择资产时提示用户选择资源。
  - 展示 Recent（最近访问/预览的资产列表入口）。

- 预览头部
  - 类型徽标与名称展示。
  - OPEN：对可打开的资源提供外部打开入口。

- 预览舞台（按类型渲染）
  - Image：图片预览。
  - Audio：音频预览（与行内播放互补）。
  - Video：视频预览。
  - 3D Model：模型预览（支持 glb/gltf/obj/stl 等，gltf 相关资源从同目录解析）。
  - Document：文档预览（pdf/txt/md/json 等）。
  - Unknown/Error：不支持或加载失败时显示错误/提示信息。

- 详情信息区
  - Name：名称。
  - File path：文件路径。
  - Resolution：分辨率（适用于图片/视频等）。
  - Format：格式信息（扩展名/类型）。

4. 底部活动日志抽屉（Activity Dock）
- 抽屉头部：Activity 标题与折叠/展开控制。
- 日志列表：按级别展示系统消息、错误与操作反馈；空态显示 Ready. 等状态提示。
